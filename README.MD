# Simply-Dash

Simply-Dash is a simple web dashbaord that displays the time, date, weather and Google calendar.

## Weather
Weather uses Openweather.com as the weather provider, you can get your free OpenWeather API key here: [OpenWeather](https://openweathermap.org/api)
